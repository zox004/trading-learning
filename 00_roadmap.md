# 학습 로드맵 — 트레이딩 직접 능력 향상 1-2년 계획

## 프로젝트 개요

### 목표
크립토 직접 트레이딩 능력 향상. 자동매매 시스템 구축이 아닌
본인의 트레이딩 판단력·실행력·심리 통제 능력 개발.

비교 기준: 보존된 외부 트레이더 165건 분석 데이터.
양방향 균형(LONG/SHORT 51:49), 짧은 보유(median 51분),
승률 80%, MFE capture 0.19 같은 패턴을 본인이 재현·발전시킬 수 있도록.

### 학습 동기
2026년 1월 비트코인 폭락 시 마틴게일 봇으로 청산 경험.
같은 함정에 두 번 빠지지 않기 위한 체계적 학습.

### 학습 자원
- 매일 4-5시간 학습 가능
- 1-2년 장기 학습 각오
- 한국어 자료 우선
- 멘토(Claude 데스크톱) + 클코(Claude Code) 협업

## 단계 1 — 시장·차트 기초 🔄

### 목표
차트를 읽을 수 있다 + 진입 시나리오를 작성할 수 있다.

구체적으로:
1. 봉 차트 구조와 거래량 해석
2. 추세·횡보·전환 식별
3. 지지·저항·박스권 그리기
4. 핵심 지표(이동평균·RSI·MACD) 사용
5. 9개 핵심 개념을 실제 차트에서 식별
6. 가상 진입 시나리오 작성 (진입가·손절가·익절가·손익비)

### 학습 자료
- 책: 『주식시장에서 살아남는 심리 투자 법칙』(엘더) 정독 계속
- 차트: TradingView (BTCUSDT 일봉·4시간봉·1시간봉)
- 노트: `book_notes/elder_trading_psychology/` (계속 누적)
- 신규 작성: `stage1_charts/market_basics.md`, `stage1_charts/chart_reading.md`

### 도구
- TradingView 무료 플랜
- 보조 자산: BTC (메인 학습), ETH (보조 참고)
- 시간프레임: 일봉 → 4시간봉 → 1시간봉 (이 순서로)

### 주차별 계획

**1-2주차: 차트 인터페이스 익히기**
- TradingView 기본 사용법 (시간프레임 전환, 지표 추가, 드로잉)
- 봉 차트 구조 (시가·고가·저가·종가, 양봉·음봉)
- 거래량 막대 이해
- BTCUSDT 일봉 차트에서 매일 30분 관찰

**3-4주차: 추세와 구조**
- 고점·저점 구조 (HH/HL, LH/LL)
- 추세 식별 (상승·하락·횡보)
- 이동평균선 (20·50·200) 도입
- 9개 핵심 개념 중 항목 1(황소-곰)·5(수평선 우선) 차트에서 식별
- 매일 BTC 일봉 추세 1줄로 기록

**5-6주차: 지지·저항·박스권**
- 수평 지지선·저항선 그리기 실습
- 역할 전환(Role Reversal) 사례 찾기
- 박스권 식별
- 9개 핵심 개념 중 항목 6(지지·저항) 차트에서 식별
- 매일 BTC 차트에서 박스권 또는 지지·저항 1개씩 식별·표시

**7-8주차: 모멘텀과 다이버전스**
- RSI 지표 도입
- MACD 지표 도입
- 약세·강세 다이버전스 식별
- 9개 핵심 개념 중 항목 3(다이버전스) 차트에서 식별
- 과거 차트에서 다이버전스 사례 5개 이상 찾기

**9-10주차: 돌파와 가짜 돌파**
- 박스권 돌파 패턴 식별
- 가짜 돌파(F) 사례 분석
- 거래량·종가·리테스트 3가지 도구 적용
- 9개 핵심 개념 중 항목 7(가짜 돌파) 차트에서 식별
- 매일 진입 시나리오 1개 작성 시작 (가상)

**11-12주차: 시나리오 작성 능력 굳히기**
- 매일 BTC 일봉 분석 + 진입 시나리오 1개
- 시나리오 형식: 진입가 / 손절가 / 익절가 / 손익비 / 근거
- 사후 검증: 1주일 후 실제 가격이 시나리오대로 갔는지 기록
- 단계 1 완료 평가

### 산출물
- `stage1_charts/market_basics.md` — 시장 기초 정리
- `stage1_charts/chart_reading.md` — 차트 도구 사용법
- `book_notes/elder_trading_psychology/` — 책 노트 계속 누적
- 매주 1회 학습 회고 (선택)

### 완료 기준
- BTC 일봉 차트를 보고 현재 추세를 1분 이내에 판단 가능
- 의미 있는 지지선·저항선을 차트에 그릴 수 있음
- 박스권 vs 추세장을 구분 가능
- 약세 다이버전스 사례를 과거 차트에서 5개 이상 식별 가능
- 가짜 돌파와 진짜 돌파를 거래량·종가로 구분 가능
- 가상 진입 시나리오 30건 이상 작성 (진입가·손절가·익절가·손익비 포함)
- 시나리오 검증 기록 누적

### 단계 1 통과 후 권장 후속
- 단계 2 진입 전 1주일 정도 본인 시나리오 30건 회고
- 어떤 패턴에서 시나리오가 잘 맞았고, 어디서 빗나갔는지 분석
- 본인 약점 파악 후 단계 2 진행

## 단계 3 — 페이퍼 트레이딩 + 일지 ⬜

### 목표
가상 자금으로 단계 1·2에서 익힌 룰을 실제 시장에서 적용.
실거래 직전 마지막 검증 단계.

### 사용 도구
- Binance Futures Demo 또는 Bybit Testnet
- TradingView 페이퍼 트레이딩 (병행 가능)

### 거래 룰 (단계 0·1·2에서 도출)
- 1-2% 룰 (단계 0)
- 사이즈-손절폭 역산 사고 흐름 (단계 0 4-5)
- 가짜 돌파 전략 B — 리테스트 진입 (단계 1)
- 추세-박스권 차등 사이징 (단계 1, 시도 가능)
- 트레이더에게서 도출한 룰 5개 (단계 2)
- 손익비 최소 1:2 이상
- 일일 손실 한도 -2%, 도달 시 그날 거래 중단

### 진행 방식
- 매일 1-3건 진입 시나리오 작성
- 그중 룰 충족하는 것만 페이퍼 진입
- 진입과 동시에 손절 주문 거래소에 설정
- 모든 거래 일지 작성 (진입 이유·결과·교훈)

### 산출물
- `stage3_paper/template.md` (현재 placeholder 채움)
- `stage3_paper/entries/` — 일일 거래 일지
- 월간 통계 분석 (승률·손익비·MDD 등)

### 완료 기준
- 페이퍼 트레이딩 100건 이상 누적
- 승률 50% 이상 또는 손익비 2:1 이상
- 일일 한도 룰 100% 준수
- 손절 룰 어긴 거래 5% 이하
- 사이즈 룰 어긴 거래 0건
- 3개월 누적 PnL 양수 (절대 기준)

### 단계 3 통과 못 한 경우
- 통계 분석으로 약점 파악
- 약점 보강 위한 추가 학습
- 단계 1·2로 부분 복귀 가능
- "페이퍼에서 못 지키는 룰을 실전에선 더 못 지킨다"는 원칙 명심

## 단계 간 이동 원칙

### 절대 건너뛰지 않는다
단계 0 없이 단계 1, 단계 3 없이 단계 4 — 모두 1월 청산의 재발 경로.

### 후퇴 가능
단계 2 진행 중 단계 1 차트 도구 부족 발견 → 단계 1로 부분 복귀 OK.
단계 3 통계가 안 좋음 → 단계 2 분석 재검토 OK.
완벽하게 모든 룰 마스터하고 다음 단계 가는 게 아니라
"다음 단계에서 학습할 수 있을 만큼"이면 진행.

### 시간 압박 금지
"빨리 단계 4 가서 돈 벌어야지" 같은 사고가 1월 청산의 시작.
12-18개월 학습 기간을 지키는 것이 가장 큰 위험 관리.

5. 멘토가 GitHub URL로 결과 검증
