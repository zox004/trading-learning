# 트레이더 스터디 — 핵심 교훈

## 비공개 정책

본 콘텐츠는 외부 트레이더 165건 실거래 분석에서 도출한 학습 교훈만 담습니다.
**구체 금액(KRW·USD)·개별 거래의 식별 가능 정보는 포함하지 않습니다.**
관련 정책은 [`01_overview.md`](01_overview.md) 참조.

## 예정 콘텐츠

다음 라운드에서 작성:
- 양방향 균형(LONG/SHORT 51:49)에서 배우는 편향 회피
- 짧은 보유(median 51분)와 sweet spot 15-60분의 의미
- 승률 80% · payoff 0.99 — 작게 자주 이기는 패턴의 본질
- MFE capture 0.19 — 가능한 수익의 19%만 잡고 빠지는 절제
- 12/19 사고와 "peak 도달 후 멈춤" 룰의 가치
- 사이즈 단계적 확대 — 검증 후 증액의 원칙

원본 통계 요약은 로컬 메모리에 보존되어 있고, 학습 시사를 정제한 콘텐츠가 여기에 추가됩니다.
