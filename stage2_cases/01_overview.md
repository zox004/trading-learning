# 트레이더 사례 — 개요

> ⚠️ 이 데이터는 신빙성이 검증되지 않은 단순 사례 자료입니다.
> 도달 목표나 모방 대상이 아니라, 비판적으로 관찰하는 학습 재료입니다.
> "왜 됐고 왜 위험한가"를 보되, 채택 여부는 본인 페이퍼 데이터로 직접 검증합니다.

## 비공개 정책

이 디렉토리는 외부 트레이더의 165건 실거래 기록을 학습 사례로 활용하는 공간입니다.
**원본 거래 로그(`all_trades.json`)와 그로부터 직접 파생된 raw 데이터는 본 저장소에 포함되지 않습니다.**

비공개 사유:
- 외부 트레이더의 거래 기록 사용 권한이 학습 목적에 한정됨
- 식별 가능한 개별 거래 시점·가격·수량의 공개 권한 없음

학습 콘텐츠에서 사용할 때의 원칙:
- 절대 금액 노출 금지 (KRW·USD 모든 단위)
- 명목가 대비 비율(%)로만 표현
- 시간 패턴(KST 시각)은 학습 가치가 있어 보존 가능
- 트레이더 식별 정보 일체 없음
