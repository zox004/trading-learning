## 1. TradingView 기본 사용법

### 1-1. 차트 종목 선택

검색 시 같은 BTC라도 거래소·계약 종류에 따라 다른 결과가 나옵니다.

| 종류 | 적합 여부 | 비고 |
|---|---|---|
| BTCUSDT Binance 현물 | ✅ 권장 | 글로벌 최대 거래량, 노이즈 적음 |
| BTCUSDT 무기한 선물 | △ | 만기 없음, 학습 가능 |
| BTC Futures (CME) | ❌ | 전통 금융, 24시간 아님 |
| BTCUSD CM Futures (월별 만기) | ❌ | 매월 차트 끊김, 장기 학습 부적합 |

학습용 메인 차트는 **만기 없는 현물 또는 무기한 선물** 권장.

### 1-2. 지표 추가

상단 "Indicators" 클릭 → 지표명 검색 → 클릭 시 추가.

단계 1 기본 셋업:
- 거래량 (Volume) — 기본 표시
- RSI (Relative Strength Index) — 14
- MACD — 12·26·9

각 지표는 차트 아래 별도 패널에 표시됩니다.

### 1-3. 호버로 값 확인

지표의 선·막대 위에 마우스를 호버하면 정확한 이름·값이 표시됩니다.
**숫자 순서가 헷갈리면 직접 호버로 확인**.

예: MACD 패널의 숫자 3개
- TradingView는 좌측부터 **히스토그램 → MACD 선 → 시그널 선** 순서

## 2. 봉(Candle) 구조 — OHLC

캔들 하나는 4가지 가격 정보를 담습니다.

| 약어 | 영어 | 의미 |
|---|---|---|
| O | Open | 시가 (그 기간 첫 거래 가격) |
| H | High | 고가 (그 기간 최고가) |
| L | Low | 저가 (그 기간 최저가) |
| C | Close | 종가 (그 기간 마지막 거래 가격) |

### 2-1. 양봉과 음봉

- **양봉 (Green)**: 종가 > 시가 (그 기간 동안 가격 상승)
- **음봉 (Red)**: 종가 < 시가 (그 기간 동안 가격 하락)

### 2-2. 몸통과 꼬리

- **몸통 (Body)**: 시가와 종가 사이 사각형
- **윗꼬리 (Upper Wick)**: 몸통 위 가는 선. 고가 표시
- **아랫꼬리 (Lower Wick)**: 몸통 아래 가는 선. 저가 표시

### 2-3. 가격 언급의 표준 — 종가

차트 분석에서 가격 말할 때 기본은 **종가(Close)**.

이유:
- 그날의 매수·매도 줄다리기 끝의 최종 합의
- 모든 지표(이동평균·RSI·MACD)가 종가 기반 계산
- "꼬리는 노이즈, 종가가 진실"

다른 가격 사용 상황:
- 고가·저가: 지지·저항선 그리기, 손절가 설정
- 시가: 갭(gap) 분석
- OHLC 전체: 정확한 캔들 정보 필요 시

## 3. 이동평균 — SMA vs EMA

### 3-1. SMA (Simple Moving Average, 단순이동평균)

N일 종가의 단순 평균. 모든 날에 동일 가중치.

예: 5일 SMA = (1일~5일 종가의 합) / 5

특징:
- 계산 단순
- 옛날 데이터도 동등하게 반영
- 반응 느림

### 3-2. EMA (Exponential Moving Average, 지수이동평균)

같은 이동평균이지만 **최근 가격에 더 큰 가중치**.

| | SMA | EMA |
|---|---|---|
| 가중치 | 모든 날 동일 | 최근일수록 큼 |
| 반응 속도 | 느림 | 빠름 |
| 노이즈 민감도 | 안정적 | 민감 |
| 신호 시점 | 늦음 | 빠름 |

### 3-3. MACD가 EMA를 쓰는 이유

MACD = 12일 EMA - 26일 EMA

EMA를 쓰는 이유: 추세 전환을 빨리 잡기 위해.
SMA로 만들면 신호가 늦어 큰 움직임 놓침.

## 4. 모멘텀(Momentum)

가격이 한 방향으로 움직이는 **힘의 강도**.

자전거 비유:
- 페달 세게 밟음 → 가속 (강한 모멘텀)
- 페달 약하게 밟음 → 감속 시작 (모멘텀 약화)
- 페달 안 밟음 → 관성으로 굴러감 (모멘텀 유지)
- 정지 → 모멘텀 0

### 4-1. 가격 vs 모멘텀

가격이 어디로 갈지(방향)만 보는 게 아니라 **얼마나 강하게 갈지(힘)** 측정.

```
시나리오 A: 가격 1000 → 1010 → 1020 → 1030 (강한 모멘텀)
시나리오 B: 가격 1000 → 1010 → 1015 → 1018 (약한 모멘텀)
```

둘 다 상승이지만 B는 곧 멈출 가능성 큼. 모멘텀이 알려주는 정보.

### 4-2. 모멘텀 지표

| 지표 | 측정 방식 | 범위 |
|---|---|---|
| RSI | 상승/하락 비율 | 0~100 |
| MACD | 이동평균 차이 | 자유 |
| 스토캐스틱 | 최근 N일 내 위치 | 0~100 |

### 4-3. 핵심 통찰

> 가격은 거짓말할 수 있지만 모멘텀은 못 한다.

[`04_book_notes/elder_trading_psychology/key_concepts.md`](04_book_notes/elder_trading_psychology/key_concepts.md) 3번 (약세 다이버전스)의 본질. 가격이 신고가 갱신해도 모멘텀이 약해지면 추세는 거짓.

## 5. MACD 상세

### 5-1. 구조

```
MACD 선 = 12일 EMA - 26일 EMA
시그널 선 = MACD 선의 9일 EMA
히스토그램 = MACD 선 - 시그널 선
```

### 5-2. TradingView 표시 순서

```
MACD close 12 26 9   [숫자1]      [숫자2]    [숫자3]
                    히스토그램    MACD 선    시그널 선
                    (빨강/초록)   (파랑)     (주황)
```

차트 아래 별도 패널. 0선을 기준으로 양수·음수.

### 5-3. 값의 단위

MACD 숫자는 **가격 단위**.
- BTC (가격 8만): MACD 값 ±수백~수천
- ETH (가격 3천): MACD 값 ±수십~수백

절대값이 아니라 **변화·방향**이 중요.

### 5-4. 4가지 신호

| 신호 | 의미 |
|---|---|
| 0선 돌파 | 큰 추세 전환 (자주 안 나옴) |
| 시그널 선 교차 | 골든/데드 크로스 (가장 흔함, 노이즈 많음) |
| 히스토그램 변화 | 모멘텀의 방향·강도 |
| 다이버전스 | 가장 강력. 추세 약화 사전 신호 |

### 5-5. MACD 약점

- **후행 지표**: 이미 일어난 움직임 따라옴
- **횡보장 신호 남발**: 박스권에서 잦은 거짓 교차
- **시간프레임 종속**: 일봉과 1분봉이 다른 신호
- **단독 사용 금지**: 거래량·지지/저항과 결합 필수

## 6. 첫 실전 분석 — BTC 일봉 약세 다이버전스 (2026-04~05)

### 6-1. 발견 경위

단계 1 첫 학습 중 BTC 일봉 차트에서 약세 다이버전스 직접 식별.
멘토 분석의 시점 오류를 본인이 직접 검증·정정한 결과.

### 6-2. 패턴 데이터

```
시점 1: 2026-04-17 부근
- 가격 고점: 약 75,000~76,000
- MACD 히스토그램: 최대치 (당시 양수 정점)

시점 2: 2026-05-07 부근
- 가격 고점: 약 82,000~83,000 (HH, 신고가 갱신)
- MACD 히스토그램: 양수지만 명확히 작아짐 (LH)

→ 약세 다이버전스 성립
가격: HH (Higher High)
모멘텀: LH (Lower High)
```

### 6-3. 사후 확인

다이버전스 발현 후 가격 흐름:
- 5월 7일: 정점 82,000~83,000
- 5월 17일: 78,412 (약 -4~5% 하락)
- MACD 히스토그램: 음수 전환, 데드 크로스 확정

**다이버전스 신호가 실제 하락으로 확인됨.**

### 6-4. 학습 시사

- 책에서 추상적으로 배운 다이버전스가 실제 차트에서 발현
- 가격 정점과 모멘텀 정점의 시간차가 추세 약화의 핵심 신호
- 다이버전스 식별 후 사후 검증까지의 흐름 체험

다만 다이버전스는 **단독 신호로 쓰지 않음**.
- 강한 추세에서는 여러 번 발생해도 추세 지속 가능
- 단독 사용 시 잘못된 진입·손실 위험
- 지지·저항·거래량·시간프레임과 결합 필수

### 6-5. 1월 청산과의 가설적 연결

`key_concepts.md` 3번에서 정리한 가설:
- 1월 폭락 직전 BTC 일봉에 약세 다이버전스 존재 가능성 높음
- 마틴게일 봇은 볼린저 밴드만 봤음
- 일봉 다이버전스 식별 가능했다면 매수 보류 가능

본인이 이번에 첫 다이버전스 식별 성공.
앞으로 일봉 MACD 다이버전스를 매일 체크하는 습관 들이면 비슷한 함정 회피 가능.
